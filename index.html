GeofenceExample

  1. Clone the project

  2. Add the following strings resource with your Google Maps Key

    <string name="google_maps_key" templateMergeStrategy="preserve">{your key goes here}</string>

NOTE: Works best from an emulator as any other application working with location can interfere with your Mock location
